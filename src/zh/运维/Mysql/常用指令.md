# Mysql一些工作中的常用指令

## 用户管理
```sql
-- 切换到mysql数据库
MySQL [(none)]> use mysql;

-- 用户查看
MySQL [mysql]> SELECT user FROM user;

-- 查看用户拥有权限
MySQL [mysql]> show grants for user;


-- 创建用户
mysql> CREATE USER 'xxx'@'%' IDENTIFIED BY 'xxxx';


-- 授权
mysql> GRANT SELECT, INSERT, UPDATE, DELETE ON 数据库.数据表 TO user@'%';

-- 删除用户

mysql>  DROP USER XXX

```
